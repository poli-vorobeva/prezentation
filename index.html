What is TypeScript

It's not another language ,it's static typed extension to JS

TypeScripts :

  • using TS we can catch errors at compile time instead of at runtime
  • our applications became more predictable and easier to debug
  • TS brings to JavaScript:classes, interfaces, constructors, access, modifiers, generics ...
